How to Install Pixi Linux

Follow this step-by-step guide to install Pixi Linux on your system

1

Download the ISO

First, download the Pixi Linux ISO file from our download page. The ISO file is approximately 2.5GB in size.

  • Go to the Download section on our homepage
  • Click on "Download ISO" button
  • Wait for the download to complete
2

Create a Bootable USB

You'll need a USB drive with at least 4GB of free space to create a bootable installation media.

For Windows users:

  • Download and install Rufus
  • Insert your USB drive
  • Select the downloaded ISO file
  • Click "Start" and wait for the process to complete

For Linux users:

  • Use dd command: sudo dd if=pixi-linux.iso of=/dev/sdX bs=4M status=progress
  • Or use tools like balenaEtcher or GNOME Disks
3

Boot from USB

Restart your computer and boot from the USB drive you just created.

  • Insert the USB drive into your computer
  • Restart your computer
  • Press the BIOS/UEFI key (usually F2, F12, Delete, or Esc) during startup
  • Select your USB drive from the boot menu
  • Choose "Pixi Linux" from the GRUB menu
4

Run the Installer

Once you've booted into Pixi Linux, you'll see the desktop with the installer icon.

  • Double-click on the "Install Pixi Linux" icon on the desktop
  • The installer will guide you through the installation process
  • Follow the on-screen instructions carefully
5

Configure Your System

During the installation, you'll need to configure several settings:

  • Language: Select your preferred language
  • Timezone: Choose your timezone
  • Keyboard Layout: Select your keyboard layout
  • User Account: Create your user account with password
  • Disk Partitioning: Choose automatic or manual partitioning
6

Complete Installation

After configuring all settings, the installer will begin copying files and installing the system.

  • Wait for the installation to complete (usually 10-30 minutes)
  • Once finished, you'll be prompted to restart
  • Remove the USB drive and restart your computer
  • Boot into your new Pixi Linux installation!

⚠️ Important Notes

  • Backup your data: Installation will format your disk. Make sure to backup important files.
  • Minimum requirements: 4GB RAM, 20GB free disk space, 64-bit processor
  • UEFI/Legacy BIOS: Pixi Linux supports both UEFI and Legacy BIOS systems
  • Secure Boot: You may need to disable Secure Boot in BIOS settings
